# RedFX

A Fabric Minecraft Mod targeting Minecraft version **26.1.2**.

## Development Setup

1.  **Build the Mod**:
    Run the following command to compile and remap the mod. This will automatically copy the remapped jar to your Minecraft mods folder (`/home/ptphong/.minecraft/mods`):
    ```bash
    ./gradlew build
    ```

2.  **Run/Debug in Development Environment**:
    Launch the Minecraft client directly:
    ```bash
    ./gradlew runClient
    ```

## Mod Information
*   **Mod ID**: `redfx`
*   **Package**: `com.drinfonty.redfx`
